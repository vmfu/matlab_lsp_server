## Quick Start

### TUI Crush Configuration (`.crush.json`)
```json
{
  "lsp": {
    "matlab": {
      "command": "python",
      "args": ["-m", "matlab_lsp", "--stdio"],
      "filetypes": ["matlab", "m"],
      "root_markers": [".matlab-lsprc.json", "start_crush.bat"]
    }
  }
}
```

### VS Code Configuration (`.vscode/settings.json`)
```json
{
  "matlab.lsp.serverPath": "matlab-lsp"
}
```

### Neovim Configuration (Lua)
```lua
require('lspconfig').matlab_lsp.setup {
  cmd = { 'python', '-m', 'matlab_lsp', '--stdio' },
  filetypes = { 'matlab', 'm' },
  root_dir = require('lspconfig').util.root_pattern('.matlab-lsprc.json')
}
```

## Advantages Over Official MathWorks LSP

| Feature | LSP MATLAB Server | Official MathWorks LSP |
|---------|-------------------|------------------------|
| **Lightweight** | ~50MB RAM | ~500MB+ RAM |
| **Startup Time** | ~1 second | ~10+ seconds |
| **Java Dependency** | None | Required |
| **Cross-Platform** | Native support | Windows only (historical) |
| **Configuration** | Simple JSON | Complex XML |
| **License** | MIT | Proprietary |
| **Installation** | pip install | MATLAB installation required |

---

## Known Limitations

- **MATLAB Dependency:** Full diagnostics require MATLAB R2020b or later
- **Standalone Mode:** Limited features without MATLAB installation
- **Editor Support:** Requires LSP-compatible editor (VS Code, Neovim, TUI Crush, etc.)

## Configuration Example

### `.matlab-lsprc.json`
```json
{
  "matlabPath": "C:\\Program Files\\MATLAB\\R2023b\\bin\\win64",
  "maxDiagnostics": 100,
  "diagnosticRules": {
    "all": true,
    "unusedVariable": true,
    "missingSemicolon": false
  },
  "formatting": {
    "indentSize": 4,
    "insertSpaces": true
  },
  "completion": {
    "enableSnippets": true,
    "maxSuggestions": 50
  },
  "cache": {
    "enabled": true,
    "maxSize": 1000
  }
}
```
